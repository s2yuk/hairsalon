# hairsalon

ヘアサロンのホームページです。
PHPにて作成しました。
サロンとお客様のコミュニケーションツールとして使えるよう、WEB上で予約、メッセージ、ヘアカタログにて最新情報の取得ができるようになっています。

ポートフォリオのため記載されている情報は実在のものではありません。
